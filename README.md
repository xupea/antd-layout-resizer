# antd-layout-resizer

A resizer for antd's Layout components

## Installation
Using [npm](https://docs.npm.com)
```bash
npm install antd-layout-resizer
```

Or via [yarn](https://yarnpkg.com/)
```bash
yarn add antd-layout-resizer
```
